---
description: Run the MARS Assessment Application
---

This workflow starts the main PySide6 desktop application for the MARS Workspace Assessment System.

Follow these steps to run the application:

1. Ensure you are in the `mars_assessment_software` root directory where the `pyproject.toml` and `uv.lock` files are located.
2. Run the application using `uv` to ensure dependencies are handled correctly.

// turbo
```bash
uv run python mars_assessment.py
```
