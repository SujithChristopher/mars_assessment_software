---
description: Run the Simple MARS Display Application
---

This workflow launches the lightweight PySide6 application for real-time sensor data visualization and basic device control.
This is useful for quickly diagnosing the MARS device without running the full assessment protocols.

Follow these steps to run the tool:

1. Ensure you are in the `mars_assessment_software` root directory.
2. Launch the application using `uv`.

// turbo
```bash
uv run python simple_mars_display.py
```
